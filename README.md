# tmpl.nvim

A boilerplate template for creating Neovim plugins with Lua.

## Usage

1. Use this repo as a GitHub template (click "Use this template")
2. Clone your new repo
3. Run `./init.sh` and follow the prompts — it will replace placeholders with your org/plugin name and make an initial commit

## Structure

```
plugin/   # Vimscript entry point
tests/    # Plenary-based test suite
scripts/  # Test runner helpers
```

## Running tests

```sh
make test
```
